# React Native Reanimated 2

## Contexte

Apprendre les animations avec Reanimated, utiliser les worklets pour séparer le thread JS du thread UI, manier les transitions, etc...

### Outils utilisées

- React Native / Expo
- Typescript
- Reanimated 2

### Lancer en local

Pour lancer l'application en local :

```bash
npm install
npm start
```
